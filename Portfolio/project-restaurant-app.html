Restaurant Management Application

COMP2000 • Android (Java) • Staff & Guest roles

JavaAndroidSQLite REST APINotifications
GitHub Repo

Overview

Built a mobile app for restaurant operations with two user types: staff can manage menu items and reservations; guests can browse the menu and book tables. Data is persisted locally (SQLite) with secure API integration.

Key features

  • Login for staff and guests
  • Local persistence using SQLite
  • REST API integration for user access
  • Notifications for reservation changes
  • Notification preferences in settings
